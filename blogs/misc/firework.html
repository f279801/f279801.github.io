ASCII art firework for the new year

So it's 2020. Let celebrate it by a firework. And if you:

  • Love your terminal
  • Are a fan of ASCII art, which I'm sure most of us are.
  • Want to make cool thing to boostup your mood every morning you open up your compture.

Then let make an ASCII art firework.

I create the img2ascii tool sometimes ago since one of my friend show me the original version of it. It was just so fancy to me, just a bit missing of some nice features:

  • Color
  • Auto scale
  • User customization

So I decided to add those features to it.

Now we can use it to make our firework.

Generate our mood

I use this image to generate our ASCII. 

./img2ascii -m fit firework-new-year-2020_src.jpg -o firework.txt

Then let test the output 

cat firework.txt

Setup our mot

If we happy with the output then let save it and setup our mood. I normally have a set of moods that I like 

mkdir -p ~/.bashrc.d/mots/
cp firework.txt ~/.bashrc.d/mots/

Then I have a list of hook scripts that I want to run to setup my environment. 

for f in $(find ~/.bashrc.d/ -maxdepth 1 -type f); do
  source $f
done

Finally, setup our /.bashrc.d/mot script to randomly display our mots. 

# Skip it if we are inside emacs
if [ -n "$INSIDE_EMACS" ]; then
 return
fi

pwd=$(dirname "${BASH_SOURCE[0]}")

declare -a pic

index=0
for item in $(ls $pwd/mots); do
        pic[$index]="$item"
        index=$(( index  + 1 ))
done
today=$(( RANDOM % ${#pic[@]} ))
cat "$pwd/mots/${pic[$today]}"

Then let's enjoy it!
